Types of Variables

  1. Global Variables
  2. Instance Variables
  3. Local Variables
  4. Constant Variables
  5. Class Variable

Global Variables

As the name suggests, these variables are accessible anywhere in a program, in any class, in any loop... anywhere. Because they dont have any limits to where they can be accessed, there use is not recommended. Their name have to start with a '$'
E.g.: $global_variable

Instance Variables

Variables declared inside a class definition with a '@' are accessible anywhere in an instance of that class. Note that two instances of that same class will have seperate copies of the instance variable
E.g.: @instance_variable

Local Variables

Local variables exist within the method or loop where they are declared. Their names should start either with a lower case or underscore
E.g.: local_variable, _LocalVariable

Constant Variables

Constant values in a class are accessible anywhere in a class.

Class Variables

These variables can be considered to have the next level of scope compared to Instance variables because, they are shared by all instances of a class. This allows multiple instances of a class communicate to each other. Their names start with a '@@'
E.g.: @@class_variable
