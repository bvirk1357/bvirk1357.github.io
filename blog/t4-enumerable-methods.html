Enumerator#group_by

Lets say we have a problem to solve about car horse power. We are given a list of horsepower numbers. We then want to put the cars in different "power" buckets where each bucket contains horsepower values in certain range. The buckets for example could be: 0-100,101-200,......
This is where an enumerable's group_by method can be used. For our problem, we can perform an operation on every element of our array and as a different result to this hash is received, a different bucket is chosen to put the element in.
Take a look at the sample code below:

irb(main):001:0> array=[50,110,70,250]
=> [50, 110, 70, 250]
irb(main):002:0> array
=> [50, 110, 70, 250]
irb(main):003:0> array.group_by{|i| i/100}
=> {0=>[50, 70], 1=>[110], 2=>[250]}


In this method, we called group_by on our array and grouped the elements by matching the ones that give the same result when divided by 100. Note that dividing by 100 here only returns the integer value. For example, '50' and '70' matched because both resulted in '0'.
